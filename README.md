# SCF-V3

Sistema containerizado utilizando Docker.

## 📦 Baixar a imagem Docker

```bash
docker pull alexhc27/ferias-ntt:v3
```

## 🚀 Executar o container

```bash
docker run -p 5000:5000 alexhc27/ferias-ntt:v3
```

## 🌐 Acessar aplicação

Após iniciar o container, acessar no navegador:

```
http://localhost:5000
```

## 🛠 Requisitos

* Docker instalado
* Acesso à internet para baixar a imagem

## 📌 Observação

Este projeto utiliza uma imagem Docker já configurada para facilitar a execução do ambiente sem necessidade de instalar dependências manualmente.
